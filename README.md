## 🛠 Fixing Common Errors On Windows 11

Windows 11 may block software installs if it does not recognize the source or if security settings are strict. Use these methods to resolve issues.

### Run As Administrator
If the installer fails to start, right-click the file and select "Run as administrator." This provides the necessary permissions for the software to modify local system settings.

### Adjust SmartScreen Settings
Windows handles unknown installers with SmartScreen. If you see a blue window stating "Windows protected your PC," follow these steps:
1. Click the "More info" link on the blue window.
2. Click the new "Run anyway" button that appears.

### Check Compatibility Mode
If the software crashes shortly after installation, use compatibility settings:
1. Right-click the NetWorx shortcut icon on your desktop.
2. Select Properties and open the Compatibility tab.
3. Check the box for "Run this program in compatibility mode for."
4. Select "Windows 10" from the dropdown menu.
5. Click Apply, then select OK.
